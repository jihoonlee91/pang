# 게임 규칙

> 초안입니다. 세부 사항은 논의 후 확정합니다.

## 기본 규칙

- 플레이어가 발사체를 쏴서 화면의 공을 맞춰 분열시키고 최종적으로 제거한다.
- 모든 공을 제거하면 스테이지 클리어, 다음 스테이지로 진행한다.
- 공에 부딪히면 즉시 사망하지 않고 HP가 일정량 감소하며, HP가 0이 되면 게임 오버된다.

## 발사체

- 발사 시 플레이어 위치에서 화면 천장까지 일직선으로 뻗어나가는 형태다.
- 공에 닿거나 천장에 닿으면 사라진다.
- 발사체가 화면에 남아있는 동안에는 재발사할 수 없다.

## 스코어

- 공을 제거할 때마다 점수를 획득한다.
- (선택) 공 크기/스테이지에 따라 점수 차등 부여

## 종료 조건

- 게임 오버: HP가 0이 되었을 때
- 클리어: 모든 스테이지의 공을 제거했을 때
